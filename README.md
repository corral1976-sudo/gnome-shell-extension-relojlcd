Retro LCD 7-Segment Clock

A GNOME Shell extension that brings the classic aesthetic of vintage digital displays to your desktop. Featuring a high-fidelity 7-segment font and a realistic LCD feel, this extension is designed for those who appreciate retro-tech aesthetics combined with modern GNOME stability.
Preview
Features

    Authentic LCD Styles: Choose between Neon Green, Vintage Amber, and the new Retro Gray (Casio-style) modes.

    Dynamic Effects: Adjustable glow intensity for neon modes and realistic shadow depth for the LCD mode.

    Flexible Positioning: Place the clock on the Left, Center, or Right of your top panel.

    Quick Access: Click the clock to open the extension settings instantly.

    Fully Customizable: Real-time adjustments for font size and 12h/24h formats.

    Modern Integration: Built with LibAdwaita for a native and seamless GNOME settings experience.

Installation
From GNOME Extensions

The recommended way to install is via the GNOME Extensions website.
Manual Installation (Development)

    Clone this repository.

    Copy the content to ~/.local/share/gnome-shell/extensions/relojlcd@carlos.

    Compile the schema:

Bash

glib-compile-schemas ~/.local/share/gnome-shell/extensions/relojlcd@carlos/schemas/

    Restart GNOME Shell (Alt+F2, type r and Enter on X11, or log out/in on Wayland).

Compatibility

Fully compatible with GNOME Shell versions: 45, 46, 47, and 48.
